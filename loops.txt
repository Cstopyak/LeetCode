 For-Loop
 definition: is a block of code that is instructed to repeat a specific number of times
 Use the for keyword to specify a for-Loop
 for(intializer; condition; increment){

 }

 ex. for( let i = 0; i < array.length; i ++){

 }

 defintion: is a block of code that is instrucred to repeat until a condition is no longer true

 while(condition){
     ....
 }

     if ( i == 5) {
